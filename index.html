Hungry Rabbit

Create a 2D matrix below by adding rows of arrays--integers only!--to mimic a garden where each array element is a plot of carrots. Behind the scenes we're going to calculate how many carrots a rabbit can eat if he starts at the center of the garden and eats his way out.

Rules: On a given turn, the rabbit will eat the carrots available on the plot that it is on, and then move up, down, left, or right, choosing the the plot that has the most carrots. If there are no carrots left on any of the adjacent plot, the rabbit will go to sleep.

Note: Answer might be buggy if the rabbit has to choose between two plots with the same number of carrots.
